# 📚 DOI 助手書籤小工具

這是一個瀏覽器書籤小工具，可以從當前網頁提取 DOI 值，
並使用以下服務之一開啟它們：

- 🧪 Sci-Hub
- 📘 LibGen SciMag
- 🔗 官方 DOI 解析器 (doi.org)

---

## 🚀 使用方法

1. 從 `bookmarklet.min.js` 複製整行代碼。
2. 創建一個新的瀏覽器書籤。
3. 將複製的代碼粘貼到書籤的 URL 欄位中。
4. 導航到包含 DOI 的頁面（例如學術出版商網站）。
5. 點擊書籤小工具。

您將被提示：

- 選擇一個 DOI（如果檢測到多個）。
- 選擇打開論文的來源。

---

## 🔍 功能說明

該腳本通過以下方式掃描當前頁面的 DOI 值：

- 讀取 `<meta>` 標籤（例如 `citation_doi`）
- 搜索頁面正文文本
- 檢查錨點（`<a>`）標籤 URL

它會移除重複項並提示您從找到的 DOI 中進行選擇，然後
使用您選擇的來源打開所選的 DOI。

---

## 🛠 開發說明

完整可讀的源代碼可在 `bookmarklet.js` 中找到。

您可以通過以下方式自定義它：

- 添加更多 Sci-Hub 鏡像站點
- 跳過提示並自動化來源選擇
- 用自定義 UI 元素替換 `prompt()`
